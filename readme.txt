== Description ==
This plugin adds a shortcode for use with the WordPress e-Commerce Plugin.
Use the [wpsc_table_price] shortcode where you want to display the table rate prices for a product.
This plugin can save you time and energy when you update your table prices.

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. use the [wpsc_table_price] shortcode where you want to display the table prices for a product
